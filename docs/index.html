הורד לאנדרואיד

איך מתקינים?

  1. לחצו «הורד לאנדרואיד»
  2. פתחו את הקובץ שהורד (מההתראות או מ«הורדות»)
  3. לחצו «התקן». ייתכן שיופיע «מקורות לא ידועים»: יש לאשר
  4. פתחו את האפליקציה ותתחילו ללמוד

יש לכם כבר את האפליקציה? עדכונים עתידיים יופיעו מתוך «הגדרות», «בדוק עדכונים».
