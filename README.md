# Daf Yomi App 📜

A premium, modern mobile application for tracking daily Daf Yomi study, designed with a focus on Hebrew localization and a high-end user experience.

## ✨ Features

- **Hebrew Calendar Integration**: Full support for the Hebrew calendar, allowing you to track your study progress by the Hebrew date.
- **Daf Yomi Study Tracker**: Easily mark pages as learned, including the ability to mark future pages in advance.
- **Daily Dashboard**: A beautiful home screen showing today's Daf, your current streak, and recent progress.
- **Sefaria Integration**: Quick access to the full text on Sefaria for every Daf.
- **Smart Notifications**: Local notifications to remind you to complete your daily study.
- **History & Analytics**: View your entire study history and maintain your learning streaks.
- **Premium RTL UI**: Designed from the ground up for Hebrew speakers with full Right-to-Left support and modern aesthetics.

## 🚀 Tech Stack

- **Framework**: [Expo](https://expo.dev/) (React Native)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- **Database**: [Expo SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/) for local persistence
- **Calendar Logic**: [@hebcal/core](https://www.npmjs.com/package/@hebcal/core)
- **Animations**: [React Native Reanimated](https://docs.expo.dev/versions/latest/sdk/reanimated/) & [Lottie/Confetti](https://github.com/danielzotti/react-native-confetti-cannon)

## 🛠️ Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Expo Go app on your mobile device (or an emulator)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npx expo start
   ```
